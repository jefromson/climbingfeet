# Climbing Feet

Website for Climbing Feet - Premium climbing gear and calendars for climbers.

## About

Climbing Feet is your source for premium climbing gear and calendars. We offer the 2-26 calendar featuring stunning climbing destinations and routes.

## GitHub Pages

This website is hosted on GitHub Pages. The site is automatically deployed from the main branch.

## Assets

- `assets/climbing_feet_logo.png` - Logo for Climbing Feet
- `assets/calendar_2_26.svg` - 2-26 Calendar image (SVG placeholder - replace with actual calendar image)

You can replace the calendar SVG placeholder with PNG, JPG, or other image formats. Just update the file extension in `index.html` accordingly.

## Local Development

Simply open `index.html` in a web browser to view the site locally.
